Picamera

A pure Python interface to the Raspberry Pi camera module

    picamera

    This package provides a pure Python interface to the Raspberry Pi camera module for Python 2.7 (or above) or Python 3.2 (or above).

    Installation

    picamera is packaged in Raspbian, so install with apt-get:

    apt-get install python-picamera or apt-get install python3-picamera

    Remember to run apt-get update first.

    See more installation options at ReadTheDocs.
